# 🚀 Kodlanabilir

**Yazılım Geliştirme | Danışmanlık**

“Hayallerinizi kodluyoruz.” sloganıyla, modern ve ölçeklenebilir çözümler sunan bir yazılım şirketiyiz. Web uygulamaları, mobil çözümler ve özel yazılımlar geliştiriyoruz.

## 🔧 Hizmetlerimiz

- **Web & Mobil Uygulama Geliştirme**  
  React, Vue, Angular ve mobil platformlar (iOS/Android) için çözüm odaklı uygulamalar.

- **API & Back‑end Sistemleri**  
  Node.js, Python (Django/Flask), Go ile performanslı servisler.

- **Bulut Mimarileri & DevOps**  
  AWS, Azure, Docker, Kubernetes kullanımında kurulum, yönetim ve otomasyon.

- **Yazılım Eğitimi & Mentorluk**  
  JavaScript, Python, modern araçlar konusunda birebir ve kurumsal eğitimler.

- **Yapay Zeka**  
  Yapay Zeka, makine öğrenimi, model geliştirme süreçleri.
